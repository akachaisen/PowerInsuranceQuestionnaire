🔗 API Endpoints Reference
Backend: server.js · Express · JSON file storage (/data/db.json บน Render)
📋 Submissions API
POST /api/submissions บันทึกข้อมูล FNA ใหม่
GET /api/submissions รายการทั้งหมด (page, limit, search, status)
GET /api/submissions/:id รายละเอียดเต็มพร้อม notes
PATCH /api/submissions/:id/status อัปเดตสถานะ
POST /api/submissions/:id/notes เพิ่มบันทึกเจ้าหน้าที่
🎯 Stats & Match API (ใหม่)
GET /api/stats สถิติรวม (total, today, byStatus, byOccupation, avgAge)
GET /api/match/:id 🆕 Match สินค้าจาก submission ID
POST /api/match 🆕 Match สินค้าจาก FNA payload โดยตรง
GET / หน้าแบบสอบถาม (index.html)
GET /admin หน้า Admin Dashboard (admin.html)
